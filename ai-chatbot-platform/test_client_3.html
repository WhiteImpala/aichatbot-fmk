Chatbot Integration Test

Setup Required:
  1. Go to your Admin Dashboard.
  2. Create a new Chatbot (or copy the ID of an existing one).
  3. Open this file (test_client.html) in a text editor.
  4. Replace YOUR_CHATBOT_ID_HERE in the script tag below with your actual Chatbot ID.
  5. Save and refresh this page.

The chat widget should appear in the bottom right corner.
